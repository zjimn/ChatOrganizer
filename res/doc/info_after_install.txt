安装完成，你现在可以开始使用了，以下是一些使用提示：

这款软件旨在提高与AI 对话的管理效率，让你能够更加轻松地进行对话的分类、整理和操作，帮助提高工作和学习的效率。

软件界面说明

1. 左侧面板：这里是一个类似树形结构的面板，你可以在这里添加和管理不同的节点。每个节点下可以关联一组对话，方便进行分类管理。
   
2. 中间展示区：这是软件的主要信息展示区。你可以看到对话内容的列表，点击标题进行排序。顶部是搜索输入框，输入关键字过滤搜索，输入&&进行同时包含多个关键词的与搜索，输入||进行包含其中一项的或搜索。单击鼠标右键弹出菜单，可以移动或复制到另外的分类中，或修改对话，可以对对话的描述，对话内容修改，也可以点击某条对话的保留按钮以删除其他的对话保留该条对话。
   
3. 详细展示：双击其中任何一项，可以打开该对话的详细记录，如果是代码或关键词会高亮标注，按Ctrl + F进行关键词查找。

4. 顶部工具栏：顶部提供了一些常用的功能，左侧是一个预设管理，可以让你编辑、新增或者删除对话预设, 这样不用每次对话开始都描述你的需求。右侧是模型管理，你可以在这里配置不同的 AI 模型，设置备注，方便后续选取调用。

5. 设置界面：你可以在这里选择不同的模型服务，并根据需要设置相关的 API 和模型。设置完毕后，可以进行测试，确保一切正常运行。

5. 日志记录：当出现错误或请求异常时，你可以查看日志，帮助调试和查找问题。日志会记录每个请求的详细信息，帮助你追踪操作。也可以点击进入高级设置，提供更多自定义选项，打开关闭各种类型的日志记录。

6. 输入区：你可以输入请求与AI模型交互，如果打开历史对话进行对话会把历史对话添加至请求。点击新的对话进行新的对话请求（按Ctrl + Shift + N 新建对话，按Ctrl + Shift + O 取消当前请求，Alt + Enter 换行），点击旁边的下拉列表切换文本或图片，目前只有openai(deepbricks)接入了文生图。


核心功能

1. 对话管理：你可以方便地查看历史对话，进行排序、搜索。通过左侧的树形结构，你可以将不同对话分类保存，避免混乱。

2. 节点和文件夹管理：支持对节点的编辑、删除及重命名。你可以根据需要，给每个节点添加备注，方便后续操作。

3. 预设管理：你可以预设常用的翻译指令或任务模板，避免重复输入。设置好预设后，下一次使用时只需输入要处理的文本即可，省去繁琐的指令设置。

4. 搜索功能：当对话内容较多时，你可以通过搜索工具快速查找特定的文本或关键词，方便快速定位需要的内容。
